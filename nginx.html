Practica 4

Documentar correctamente el proceso de instalación y configuración de un servidor Nginx que dispone de dos sitios web:

  • Creas el fichero de configuracion asociado dentro de site-avaliable, para ello copiamos el archivo default

    cd /etc/nginx/sites-avaliables
    sudo cp default miSitio



  • Cambiamos el fichero de configuracion para poner que se escuche en el puerto 82 y la direccion sea /var/www/miSitio

    sudo nano miSitio



  • Creamos el directorio con el contenido de nuestro sitio y lo habilitamos haciendo manualmente un enlace simbolico de la carpeta sites-enabled con sites-avaliable


    sudo ln -s /etc/nginx/sites-available/miSitio /etc/nginx/sites-enabled
